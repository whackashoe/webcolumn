WebColumn


A tiny alternative to heavy frameworks such as Foundation or Bootstrap providing only a simple, customizable grid layout.
Just select the grid you'd like to use above, the examples below are in twelve grid format. Once the code is added to your stylesheet just create a row and populate it with columns. Inside these columns you can nest more rows. So the following is essentially equivalent:
		<div class="row">
		    <div class="six column">Half</div>
		    <div class="six column">
		        <div class="row">
		             <div class="six column">Quarter</div>
		             <div class="six column">Quarter</div>
		        </div>
		    </div>
		</div>
		    							
		<div class="row">
		    <div class="six column">Half</div>
		    <div class="three column">Quarter</div>
		    <div class="three column">Quarter</div>
		</div>
You may nest as much as needed, however, it may be simpler to start with a larger grid amount if you assume this to be the case. Examples based on a twelve grid are below for a better understanding.
